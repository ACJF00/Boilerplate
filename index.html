Votre carte d'urgence personnalisée

Voilà pourquoi nous avons créé la carte d'urgence Protectoo pour tous : Parfois la vie ne tient qu'à un fil, nous voulions une solution pour pouvoir prévenir un proche rapidement et informer les premiers secours sur les éléments importants pour notre vie, et sans technologie complexe : Nous avons créé la carte d'urgence Protectoo, pour tous.

En savoir plus

La carte d'urgence Protectoo est une carte d'urgence connectée qui permet de prévenir un proche et informer les secours en cas d'urgence. Elle est équipée d'un QR code qui, une fois scanné, permet d'accéder à toutes les informations importantes pour les secours.

Comment ça marche ?

J’approche ma carte d’un téléphone NFC

ou j’utilise le QR code

Je saisi le code que j’ai reçu par SMS ou Mail

Un numéro a 6 chiffres est envoyé pour «déchiffrer » le profil et un mail est automatiquement envoyé à ma personne de contact.

Je saisi le numero secret de ma carte

Il se trouve en bas du verso de ma carte

Ma page profil apparait

Mon profil est visible et les secours disposent des informations nécessaires à ma prise en charge.

Sans contact

Grâce à la puce NFC intégrée dans votre carte, posez-la simplement sur le téléphone pour ouvrir instantanément votre profil !

QR code

Également présent sur votre carte, le QR code peut être rapidement scanné pour accéder à votre profil !
